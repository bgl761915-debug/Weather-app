# 🌤️ Weather App — Animated, No Login, No API Key

A modern Python weather app that uses the **Open‑Meteo API**:

- ✅ No login
- ✅ No account
- ✅ No API key
- ✅ Free and public

Features:

- 🔥 Animated weather icons (GIFs)
- 🌗 Dark / Light mode toggle
- 🖼️ Background image changes with the weather
- 🎨 Rounded, card‑style UI
- ⚡ Loading indicator while fetching
- 🌍 City search with automatic geocoding

---

## ✨ What it shows

For any city:

- City + country
- Temperature (°C)
- Weather description
- Wind speed
- Weather code (from Open‑Meteo)
